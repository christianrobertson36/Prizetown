# Prizetown

Custom competition website app.

## Local dev

```powershell
cd C:\Users\Windows11\Documents\GitHub\Prizetown
```

API:
```powershell
cd api
npm install
npm run dev
```

Web:
```powershell
cd web
npm install
npm run dev
```
